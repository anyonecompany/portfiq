# AI 위험 평가서 (AI Risk Assessment)

**서비스명**: 포트픽 (Portfiq)
**평가일**: 2026년 3월 13일
**근거법**: 인공지능 발전과 신뢰 기반 조성 등에 관한 기본법 (2026.01.22 시행)
**평가 버전**: v2.0.0

---

## 1. 서비스 개요

포트픽은 AI 기반 ETF 뉴스 분석 및 브리핑 서비스입니다. 이용자가 등록한 ETF를 기반으로 관련 뉴스를 수집하고, Google Gemini API를 활용하여 뉴스를 분석/요약/번역하여 맞춤형 브리핑을 제공합니다.

### 핵심 기능

| 기능 | 설명 |
|------|------|
| 아침 브리핑 | 간밤 미국 시장 결과를 3줄로 요약 |
| 뉴스 영향도 분석 | 각 뉴스의 ETF 영향(호재/위험/중립) 평가 |
| 오늘 밤 체크포인트 | 주목할 시장 이벤트 안내 |
| 섹터/매크로 분석 | 포트폴리오 섹터 집중도 및 매크로 민감도 시각화 |

### 사용 AI 모델

- **모델**: Google Gemini API
- **접근 방식**: API 호출 (자체 모델 학습 없음)
- **데이터 학습**: 이용자 데이터로 모델을 학습하지 않음
- **입력 데이터**: 뉴스 원문 텍스트, ETF 티커 심볼
- **출력 데이터**: 한국어 번역문, 요약 브리핑, 영향도 등급

## 2. 고영향 AI 해당 여부

### 판단 결과: 비해당 (일반 AI)

| 판단 기준 | 해당 여부 | 근거 |
|----------|----------|------|
| 생명/신체 안전 영향 | 비해당 | 뉴스 요약 및 정보 제공 서비스로, 직접적 안전 영향 없음 |
| 기본권 영향 | 비해당 | 개인의 권리/자유를 제한하는 의사결정에 관여하지 않음 |
| 불공정 차별 가능성 | 비해당 | 특정 집단을 대상으로 차별적 결정을 내리지 않음 |
| AI 직접 의사결정 | 비해당 | 정보 제공만 수행하며, 투자 결정/실행을 하지 않음 |
| 금융 의사결정 관여 | 비해당 | 신용평가/대출 심사 등 금융 의사결정이 아닌, 뉴스 정보 제공에 한정 |
| 자동 거래/투자 결정 | 비해당 | 매매 주문, 자동 리밸런싱 등 거래 기능 없음 |

**결론**: 이용자가 정보를 참고하여 스스로 투자 판단을 내리는 구조이므로 고영향 AI에 해당하지 않습니다. 생성형 AI 서비스로서 투명성 확보 의무를 준수합니다.

## 3. 위험 식별

### 3-1. 식별된 위험

| 위험 ID | 위험 유형 | 설명 | 발생 가능성 | 영향도 | 위험 등급 |
|---------|----------|------|-----------|--------|----------|
| R-01 | AI 분석 오류 | AI가 뉴스 원문을 잘못 번역하거나 핵심 내용을 누락/왜곡 | 중 | 중 | 중 |
| R-02 | 잘못된 영향도 평가 | 뉴스의 ETF 영향을 호재/위험으로 잘못 분류 | 중 | 중 | 중 |
| R-03 | 투자자 오인 가능성 | 이용자가 AI 분석을 확정적 투자 조언으로 오인 | 중 | 높음 | 중 |
| R-04 | 환각(Hallucination) | AI가 존재하지 않는 사실이나 데이터를 생성 | 낮음 | 높음 | 중 |
| R-05 | 편향된 분석 | 특정 방향(긍정/부정)의 뉴스를 과대 또는 과소 반영 | 낮음 | 중 | 낮음 |

### 3-2. 위험에 해당하지 않는 항목

- **개인정보 유출**: 직접적 개인 식별 정보를 수집하지 않으며, 개인정보는 AI에 전달되지 않음
- **딥페이크**: 이미지/영상/음성 생성 기능 없음
- **자동 거래 실행**: 투자 주문이나 거래를 실행하는 기능 없음

## 4. 완화 조치

### R-01: AI 분석 오류

- 원문 뉴스 링크를 항상 함께 제공하여 이용자가 직접 확인 가능
- 원문 헤드라인과 번역본을 동시 표시
- AI 생성 콘텐츠에 "AI 분석" 라벨 표시

### R-02: 잘못된 영향도 평가

- "투자 조언이 아닌 참고 정보" 면책 고지 상시 노출
- AI 생성 콘텐츠에 "AI 분석" 라벨 표시
- 영향도 분류는 참고용임을 명시

### R-03: 투자자 오인 가능성

- 앱 전반에 "투자 조언이 아닌 참고 정보" 면책 고지 상시 노출
- 이용약관에 투자 면책 조항 명시
- 설정 화면에 AI 서비스 고지 문구 포함
- mock 데이터 등 시뮬레이션 결과는 명확히 표시

### R-04: 환각(Hallucination)

- 뉴스 원문 기반으로만 분석 수행 (근거 없는 정보 생성 방지)
- 원문 링크를 제공하여 이용자의 교차 검증 지원
- 사실 관계가 중요한 수치/날짜는 원문에서 직접 추출
- mock 데이터 사용 시 해당 사실을 명확히 표시

### R-05: 편향된 분석

- 다양한 뉴스 소스로부터 수집하여 편향 최소화
- 중립적 분석을 유도하는 프롬프트 설계 적용
- 긍정/부정/중립 뉴스를 균형 있게 노출

## 5. 편향성 테스트

### 테스트 방법

ETF 분석은 이용자가 직접 등록한 ETF를 기준으로 수행되므로, 특정 종목이나 섹터에 대한 시스템적 편향이 발생하지 않는 구조입니다.

### 검증 항목

| 항목 | 결과 | 비고 |
|------|------|------|
| 특정 ETF 편향 | 없음 | 이용자 등록 기반으로만 분석 수행 |
| 긍정/부정 편향 | 없음 | 중립적 프롬프트 설계 적용 |
| 뉴스 소스 편향 | 없음 | 다양한 소스에서 뉴스 수집 |
| 섹터 편향 | 없음 | 이용자 포트폴리오 기반 분석 |

## 9. 결론

포트픽은 금융 정보 분석 서비스로서 **고영향 AI에 해당하지 않는 일반 AI**로 분류됩니다. 다만, 생성형 AI 서비스로서 「인공지능기본법」에 따른 투명성 확보 의무를 준수합니다.

주요 완화 조치 요약:
- "투자 조언이 아닌 참고 정보" 면책 고지 상시 노출
- 모든 AI 생성 콘텐츠에 "AI 분석" 라벨 표시 (AI 라벨링)
- 원문 뉴스 링크 제공으로 이용자의 교차 검증 지원 (설명가능성)
- mock 데이터 사용 시 명확한 표시
- 다양한 뉴스 소스 및 중립적 프롬프트로 편향 최소화
- 설정 화면 및 이용약관에 AI 사용 사실 명시 (투명성)
- 개인정보는 AI에 전달되지 않음
